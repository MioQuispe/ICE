# CLI Commands

ICE provides a set of CLI commands to manage your Internet Computer project.

## Core Commands

- `ice init <project-name>`: Initialize a new ICE project
- `ice install <canister-name>`: Install a canister from npm
- `ice update <canister-name>`: Update a canister to the latest version
- `ice uninstall <canister-name>`: Remove a canister from your project
- `ice run <task-name>`: Run a defined task
- `ice generate <type>`: Generate boilerplate code

## Build and Deploy

- `ice build`: Build your project
- `ice deploy`: Deploy your project to the Internet Computer

## Development

- `ice dev`: Start a development server
- `ice test`: Run your project's tests

## Utility Commands

- `ice info`: Display information about your project
- `ice clean`: Clean build artifacts
- `ice help`: Display help information

For detailed information on each command, use `ice help <command>`.
