# CLI Commands

Crystal provides a set of CLI commands to manage your Internet Computer project.

## Core Commands

- `crystal init <project-name>`: Initialize a new Crystal project
- `crystal install <canister-name>`: Install a canister from npm
- `crystal update <canister-name>`: Update a canister to the latest version
- `crystal uninstall <canister-name>`: Remove a canister from your project
- `crystal run <task-name>`: Run a defined task
- `crystal generate <type>`: Generate boilerplate code

## Build and Deploy

- `crystal build`: Build your project
- `crystal deploy`: Deploy your project to the Internet Computer

## Development

- `crystal dev`: Start a development server
- `crystal test`: Run your project's tests

## Utility Commands

- `crystal info`: Display information about your project
- `crystal clean`: Clean build artifacts
- `crystal help`: Display help information

For detailed information on each command, use `crystal help <command>`.
